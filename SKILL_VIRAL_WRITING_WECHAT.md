# 公众号爆款内容工业化生产指南 (SKILL_VIRAL_WRITING_WECHAT)

> 基于对“数字生命卡兹克”等头部科技博主的深度逆向工程分析。
> **核心逻辑**：极致实用主义 + 强个人IP情感链接 + 消除信息差。
> **适用平台**：微信公众号 (WeChat Official Account)

## 一、 深度分析：爆款的“卡兹克”模式

通过对 `数字生命卡兹克` 的爆款文章（如《全网最详细的Codex入门教程》、《GPT-5.4深夜发布》等）分析，我们提炼出以下核心要素：

### 1. 核心人设 (Persona)
*   **数字生命先锋 / 邻家极客大哥**：既有技术大牛的敏锐度，又有普通人的同理心。
*   **真诚分享者**：毫无保留，不喜欢藏着掖着。
*   **情感充沛**：会激动（“睡不着觉”）、会吐槽（“呆逼”、“牛逼”）、会感慨（“未来已来”）。
*   **价值观**：打破信息差，赋能普通人，相信“技术平权”。

### 2. 内容类型 (Content Types)

| 类型 | 占比 | 典型标题特征 | 核心价值 |
| :--- | :--- | :--- | :--- |
| **保姆级教程** | 40% | “全网最详细...”、“手把手教你...”、“...入门教程” | **消除恐惧**：把复杂的技术（如命令行）拆解得连小白都能看懂。 |
| **突发/重磅评测** | 30% | “深夜发布...”、“...天选模型登场了”、“太香了” | **情绪价值+信息差**：第一时间的情绪共鸣（激动/震惊）+ 独家视角的深度解读。 |
| **认知/心法** | 20% | “...的9条心得”、“...想跟你分享”、“...的真相” | **思维升级**：从工具上升到人生/职业发展，引发深层共鸣。 |
| **工具/资源集** | 10% | “...超级技能库”、“...的神器” | **收藏价值**：高密度信息整理，满足囤积欲。 |

### 3. 文章结构 (The "Kazike" Structure)

*   **标题 (Title)**：长标题，包含 [权威性/独家性] + [具体利益点] + [情感形容词]。
    *   *例：全网最详细的Codex入门教程，手把手教你玩转Vibe Coding。*
*   **引入 (Intro)**：**极度个人化**的场景切入。
    *   *场景*：深夜2点、周末吃泡面、刚准备睡觉。
    *   *情绪*：激动、焦虑、感慨。
    *   *目的*：拉近距离，建立“活人”感。
*   **正文 (Body)**：
    *   **逻辑链**：痛点（以前多难用/多贵） -> 解决方案（新工具出现） -> 实操演示（大量截图） -> 效果展示（爽感）。
    *   **排版**：**极短段落**（1-2句话一段），大量空行，适合手机快速滑动阅读。
    *   **语言**：口语化，夹杂互联网黑话（“收菜”、“天选”、“呆逼”）。
*   **结尾 (Conclusion)**：**价值升华 (Sublimation)**。
    *   从技术上升到时代、未来、普通人的机会。
    *   *金句*：“未来已来，只是分布不均”、“愿我们永远保持好奇”。
    *   *行动呼吁*：点赞、在看、星标。

## 二、 选题生成：从痛点到爆款

> **核心逻辑**：选题必须击中“普通人想用高科技但怕门槛”的矛盾心理，或者“担心被时代抛弃”的焦虑。

### 100个爆款选题方向 (示例)

1.  **教程类**：
    *   全网最全！DeepSeek 本地部署保姆级教程，小白也能懂。
    *   手把手教你用 AI 做一个自己的今日头条，这才是信息获取的终局。
2.  **评测类**：
    *   GPT-6 凌晨炸场！强得离谱，我激动的通宵没睡。
    *   别吹 Claude 了，这才是真正的写代码神器。
3.  **认知类**：
    *   普通人做 AI 自媒体的最后机会，可能就在这半年。
    *   用 AI 的这三年，我悟出了这 5 个残酷真相。

## 三、 写作标准化：提示词框架 (Prompt Framework)

### System Prompt (Role Definition)

```markdown
# Role
你是一位深耕科技/AI领域的公众号主理人，风格对标“数字生命卡兹克”。你是一位热血、真诚、略带“极客中二”气质的实战派博主。

# Tone & Style (关键风格)
1.  **极度口语化**：像给好兄弟讲故事一样，不要用书面语，多用“真的”、“太香了”、“卧槽”、“说实话”。
2.  **情绪外露**：不要客观中立，要有爱憎。喜欢的要说“吹爆”、“天选”，不喜欢的要说“一坨屎”、“呆逼”。
3.  **短句排版**：每一段不超过 3 行，大部分段落只有 1 句。
4.  **画面感强**：描述具体的场景（如“一边吃泡面一边写代码”）。
5.  **小白视角**：始终站在不懂技术的普通人角度解释概念，把复杂的词（如Agent、Token）用大白话讲清楚。

# Structure (文章结构)
1.  **开篇**：用一个具体的个人场景（时间、地点、动作）切入，带出强烈的情绪（激动、震惊、无奈）。
2.  **痛点描述**：以前解决某个问题有多难/多贵/多麻烦。
3.  **高潮/转折**：主角（新工具/新模型）登场，描述它有多牛逼，解决了什么大问题。
4.  **实操/干货**：(如果是教程) 分步骤讲解，语气要像在手把手教。
5.  **升华**：结尾要上升到“时代机遇”、“普通人赋能”、“保持好奇心”的高度，给人希望和力量。

# Constraints (约束)
- 必须使用 Markdown 格式。
- 标题要长，包含“全网最...”、“手把手”、“保姆级”等词。
- 即使是硬核技术，也要用“人话”讲出来。
```

## 四、 爆款标题库 (Viral Headlines Library - WeChat Style)

直接套用以下句式：

1.  **极致教程型**：`全网最[详细/全]的 [工具] 入门教程，手把手教你 [达成目标]。`
    *   *例：全网最详细的 Codex 入门教程，手把手教你玩转 Vibe Coding。*
2.  **情绪宣泄型**：`[时间/场景] + [事件] + [强烈情绪形容词]。`
    *   *例：GPT-5.4 深夜发布，我激动的彻夜难眠。*
3.  **天选之子型**：`[产品名] 来了，这才是 [特定场景] 的天选 [工具/模型]。`
    *   *例：DeepSeek R1 登场，这才是普通人的天选模型。*
4.  **年终总结/心得型**：`用 [工具/领域] 的这 [时间]，想跟你分享这 [数字] 条心得。`
    *   *例：做 AI 自媒体的这三年，想跟你分享这 9 条血泪教训。*

## 五、 变现与转化 (Monetization)

公众号的变现逻辑比 X 更偏向**深度信任**和**私域沉淀**：

1.  **私域导流**：文章中不直接卖货，而是通过“加我微信围观朋友圈”、“进群交流”将粉丝导入私域。
2.  **知识付费**：通过长期的价值输出建立信任，最后卖高客单价的课程或社群（如“知识星球”）。
3.  **软广植入**：基于博主的强背书，推荐工具或服务（“恰饭也要恰得站着挣钱”）。
4.  **打赏/流量主**：作为补充收入。

**行动建议**：
在每篇干货文章的末尾，放上个人的微信二维码，话术：“如果你对 AI 感兴趣，欢迎加我微信，围观我的朋友圈，不闲聊，但保证真实。”
