I mean, 2026年了，还有人在忍受“截图-保存-上传-复制-粘贴”这种反人类流程？
**It's just unnecessary.** 真的。

写文章最怕什么？是 **打断 (Interruption)**。
你的灵感很贵，不要浪费在上传图片这种无聊的重复劳动上。
这真的不是人干的事。

所以我手搓了一个 VS Code 插件：**GitHub Paste Plugin**。

👇 效果很简单 (Simple & Sharp)：
你在 Markdown 里直接 **Ctrl + V**。
**Boom 💥**
图片、视频、音频、PDF... **Everything is linkable.**
自动上传到 GitHub，秒变 jsDelivr 的 CDN 加速链接。

✅ **全能 (All-in-One)**：截图、录屏、文档，想贴什么贴什么。
✅ **免费 (Free)**：GitHub 仓库当存储，白嫖到底。
✅ **极速 (Fast)**：jsDelivr 全球 CDN 加速，秒开。
✅ **心流 (Flow)**：没有任何弹窗和废话，Just keep flowing.

别问为什么要做这个。

我不想把时间浪费在那些没有产出的事情上。
