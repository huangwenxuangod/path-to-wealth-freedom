# GitHub Paste Plugin (全能文件粘贴助手)

> **定位**：VS Code 里的“万能粘贴键”。
> **核心卖点**：**Everything is linkable.** 截图、视频、音频、PDF... 复制 -> 粘贴 -> 生成 CDN 链接。
> **适用场景**：技术博客、播客脚本、资源分享、全格式文档编写。

## 📂 文件夹结构
*   `教程/`: 存放插件的使用教程、配置指南。
*   `推广/`: 小红书、X (Twitter) 等社交媒体的推广文案。
*   `素材/`: 存放演示 GIF、截图。
