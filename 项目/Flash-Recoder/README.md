# Flash-Recoder (Auto Screen Recorder)

> **定位**：你的技术护城河 & 内容生产工具。
> **核心卖点**：全自动后台录制、无感运行、Time-lapse 生成器。
> **Slogan**：记录你创造的每一行代码，让时间看得见。

## 📂 文件夹结构
*   `Demo/`: 存放你自己使用 Recoder 录制的开发过程视频。
*   `原理/`: 存放架构图、技术实现笔记。
*   `发布/`: 存放发布到 X/B站 的视频素材。
