# 「点→看→赞→评」爆款全链路 SOP（从看到到输出）

## 0. 一句话核心逻辑
用**强动机**让人点 → 用**强情绪/强新知**让人看完 → 用**爽点/共鸣**让人点赞 → 用**冲突/疑问**让人评论。

---

## 1. 看到（收集素材，不写流水账）
只抓 3 个东西：
1. 标题/封面
2. 开头 3 秒
3. 评论区热评前三

用一句话记：
**我为什么点？为什么看完？为什么想赞/评？**

---

## 2. 拆解（把爆款拆成四个“动机”）

### A. 点（点击动机）
三种：**好奇缺口 / 利益相关 / 情绪戳中**

可复用判断句：
- 这是话只说一半？
- 这跟我的钱/时间/成长有关？
- 这是愤怒/爽/委屈/共鸣/颠覆？

### B. 看（完读动机）
关键不是长短，是**认知成本**。

结构模板：
开头 3 秒：结论 / 反差 / 痛点  
中段：每段只讲一个信息  
节奏：痛点 → 原因 → 方法 → 结果  

### C. 赞（点赞动机）
三类：**共鸣 / 干货 / 颠覆**

判断句：
- 这句话把我情绪接住了？
- 这东西我立刻能用？
- 我原来一直错了？

### D. 评（评论动机）
四类：**情绪输出 / 求答案 / 反驳 / 补充**

评论区必备：
**立场 + 冲突 + 留白 + 提问**

---

## 3. 转译（把爆款结构换成你的观点）
只留一个核心结论，然后把四步套进去：

**标题（点）**：疑问 / 反差 / 结果  
**开头（看）**：3 秒内直接给结论/痛点  
**中段（赞）**：给一个被理解/被颠覆的情绪钩子  
**结尾（评）**：留一个没说满的空白或冲突  

---

## 4. 产出（可直接发布的文案骨架）
标题：  
“____ 为什么 90% 的人都做错了？”

开头：  
“一句话结论：____。”

中段：  
“如果你也____，你一定懂这句话。”  
“大多数人错在____，真正的解法是____。”

结尾（引评）：  
“但这点我还没讲完，你觉得____？”

---

## 5. 验收（数据反推问题）
只看四个指标：
点击率、完读率、点赞率、评论率。

哪个低，就回到对应环节重写：
点低 → 标题  
看低 → 开头/节奏  
赞低 → 情绪钩子  
评低 → 冲突/留白

---

一句话总结：
**用好奇让你点，用简单让你看，用情绪让你赞，用留白让你评。**
